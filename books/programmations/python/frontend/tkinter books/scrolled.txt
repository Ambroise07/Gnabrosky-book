Bonjour ! En Tkinter, pour détecter le scroll vers le haut de la souris, tu dois lier l'événement 
de la molette à un widget (comme une Canvas, un Frame, etc.).

Le comportement dépend du système d'exploitation :
✅ 1. Sous Windows et Linux :

    <MouseWheel> est l'événement utilisé.

    Le sens du scroll est donné par event.delta :

        Scroll vers le haut → event.delta > 0

        Scroll vers le bas → event.delta < 0

✅ 2. Sous macOS :

    Le même événement <MouseWheel> est utilisé.

    Mais la valeur de event.delta est généralement beaucoup plus petite, souvent +1 ou -1.


import tkinter as tk


def on_mouse_wheel(event):
    
	if event.delta > 0:
        
		print("Scroll vers le haut")
    
	else:
        
		print("Scroll vers le bas")



root = tk.Tk()
root.geometry("300x200")


frame = tk.Frame(root, width=300, height=200, bg="lightgray")

frame.pack()



# Liaison de l'événement à la molette de la souris

frame.bind("<MouseWheel>", on_mouse_wheel)  # Windows et macOS

frame.bind("<Button-4>", lambda e: print("Scroll vers le haut (Linux)"))

frame.bind("<Button-5>", lambda e: print("Scroll vers le bas (Linux)"))


root.mainloop()
